## 💻 How to Use

### Simple: Add CSS Classes

```jsx
// Beautiful button
<button className="btn btn-primary hover-glow animate-scale-in">
  Click me
</button>

// Animated card
<div className="card hover-lift animate-fade-in-up">
  Content
</div>

// Staggered grid
<div className="grid grid-3 grid-stagger">
  {items.map(item => (
    <div key={item.id} className="card-animated">
      {item.content}
    </div>
  ))}
</div>
```

### Advanced: Combine Effects

```jsx
<div className="card hover-lift animate-fade-in-up delay-200">
  {/* Lifts on hover + fades in + 200ms delay */}
</div>
```


## 🎨 Customization

To adjust animation speeds globally, edit `:root` in `index.css`:

```css
:root {
  --transition-fast: 150ms;   /* Change from 150ms */
  --transition-base: 300ms;   /* Change from 300ms */
  --transition-slow: 500ms;   /* Change from 500ms */
}
```
